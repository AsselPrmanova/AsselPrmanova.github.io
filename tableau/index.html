Superstore Analysis - Tableau

This dashboard provides an interactive overview of sales performance for a retail dataset.

Project Overview

  • KPI Cards: Total Sales, Total Profit, Number of Orders, Number of Customers, Number of Returns
  • Dynamic Parameter: Switch between Sales, Profit, and Orders to update all visualizations
  • Visualizations: Map by State, Segment Analysis, Time Trends, Top 5 Sub-categories, Returns by Managers
  • Filters: Date, Ship Mode, Product
